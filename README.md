# Scioly-Robot-Tour
Reply to the email with your github account username (or create one and then reply). Create branches and PR requests if you want to edit the code. 

To actually use it you have to 
1) install vs code if you don't have
2) install the latest python
3) install ev3 extension
4) enter .venv environment with either py -3 -m venv .venv for windows or python3 -m venv .venv for mac
5) pip install pybricks-stubs
6) restart terminal

Everything should work after that I think
